Public Mobile Phone Plan Picker

REACT JAVASCRIPT HTML5 CSS3 RWD GIT

A mockup of a phone plan picker web app made with reference to Public Mobile's phone plans. This application is created using React as a framework and working with vanilla JavaScript. It is categorized into two main components, one to show the plan options and one to display the price. When a plan option is selected the application's state is updated to store the price of the selected option. The application then calculates the plan price, determines if any discounts are applicable, and then displays the plan value, discount value, and total cost of the plan after subtracting the discount.
